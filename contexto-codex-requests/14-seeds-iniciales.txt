3. Supervisor

ESTADOS

1. PENDING / Pendiente / SortOrder 1 / IsFinal 0
2. IN_REVIEW / En revisión / SortOrder 2 / IsFinal 0
3. APPROVED / Aprobada / SortOrder 3 / IsFinal 0
4. IN_PURCHASE / En compra / SortOrder 4 / IsFinal 0
5. READY_TO_DELIVER / Lista para entregar / SortOrder 5 / IsFinal 0
6. PARTIALLY_DELIVERED / Entrega parcial / SortOrder 6 / IsFinal 0
7. DELIVERED / Entregada / SortOrder 7 / IsFinal 1
8. REJECTED / Rechazada / SortOrder 8 / IsFinal 1
9. CANCELLED / Cancelada / SortOrder 9 / IsFinal 1
