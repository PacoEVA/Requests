# Guía de uso para empleados

Esta guía explica cómo utilizar el portal de empleados para crear y dar seguimiento a requisiciones de material gastable.

## 1. Identificación del empleado

El portal de empleados no utiliza un inicio de sesión tradicional con usuario y contraseña.

La primera vez que entre al sistema, deberá completar sus datos de identificación:

- Nombre
- Departamento
- Código de empleado, si aplica
- Teléfono o extensión, si aplica

Después de identificarse, el sistema recordará su información en ese navegador. Esto permite cerrar la página y volver más tarde sin tener que registrarse nuevamente.

Si cambia de computadora, navegador o borra los datos del navegador, es posible que tenga que identificarse otra vez.

## 2. Pantalla principal

Al entrar al portal verá un resumen con accesos rápidos para:

- Crear una nueva requisición
- Ver sus requisiciones
- Actualizar su perfil

También podrá ver sus requisiciones recientes y entrar al detalle de cada una.

## 3. Crear una requisición

Para solicitar materiales:

1. Entre a **Nueva requisición**.
2. Seleccione la prioridad de la solicitud.
3. Agregue los materiales que necesita.
4. Indique la cantidad solicitada para cada material.
5. Si el material no aparece en el catálogo, puede escribirlo de forma manual.
6. Agregue comentarios si necesita aclarar algo.
7. Presione **Crear requisición**.

El sistema no permite enviar una requisición vacía. Cada línea debe tener una cantidad mayor que cero.

## 4. Materiales de catálogo y materiales manuales

Puede solicitar materiales de dos formas:

- **Material de catálogo:** cuando el material ya existe en el listado del sistema.
- **Material manual:** cuando necesita pedir algo que no aparece en el catálogo.

En ambos casos debe indicar la cantidad solicitada.

## 5. Consultar mis requisiciones

En **Mis requisiciones** puede ver el listado de solicitudes que ha creado.

Cada requisición muestra información como:

- Código de requisición
- Fecha
- Prioridad
- Estado

Para ver más detalles, seleccione el código de la requisición.

## 6. Estados de una requisición

Los estados indican en qué punto se encuentra la solicitud.

Los estados principales son:

- **Pendiente:** la requisición fue creada y está esperando revisión.
- **En revisión:** Compras está evaluando la solicitud.
- **Aprobada:** la solicitud fue aprobada.
- **En compra:** los materiales están siendo gestionados.
- **Lista para entrega:** los materiales están disponibles para entregar.
- **Entrega parcial:** se entregó una parte de lo solicitado.
- **Entregada:** la requisición fue completada.
- **Rechazada:** la solicitud no fue aprobada.
- **Cancelada:** la solicitud fue cancelada.

Cuando una requisición cambia de estado, podrá verlo en el detalle de la solicitud.

## 7. Cancelar una requisición

Puede cancelar una requisición propia mientras todavía no esté en un estado final.

No podrá cancelarla si ya fue:

- Entregada
- Rechazada
- Cancelada

Al cancelar, debe indicar un motivo.

## 8. Comentarios

Dentro del detalle de una requisición puede enviar comentarios.

Use los comentarios para:

- Aclarar una solicitud
- Responder preguntas de Compras
- Dar información adicional sobre el material solicitado

Los comentarios se actualizan en tiempo real dentro de la plataforma.

## 9. Notificaciones

El sistema puede mostrar notificaciones cuando:

- Cambia el estado de una requisición
- Recibe un comentario nuevo
- Se registra una entrega

Si el navegador solicita permiso para mostrar notificaciones, puede aceptarlo para recibir avisos aunque no esté mirando directamente la pantalla del sistema.

## 10. Actualizar perfil

En **Perfil** puede actualizar sus datos de identificación.

También puede reiniciar su identidad del navegador si necesita volver a identificarse desde cero.

## 11. Recomendaciones de uso

- Revise bien las cantidades antes de enviar una solicitud.
- Use comentarios claros cuando el material sea manual o necesite explicación.
- Consulte el estado de sus requisiciones desde **Mis requisiciones**.
- No cree una nueva requisición si solo necesita agregar información a una existente; use los comentarios.
